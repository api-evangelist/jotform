# Jotform (jotform)

Jotform is an online form builder with strong workflow, payments and data collection capabilities. The Jotform API is exposed across three regional / compliance endpoints — Standard (US), EU and HIPAA — covering forms, submissions, reports, folders, files, system, user and webhooks. APIKEY header or query authentication.

**APIs.json:** [apis.yml](apis.yml)

## APIs
- **Standard / US** — `https://api.jotform.com` — primary REST API.
- **EU** — `https://eu-api.jotform.com` — EU data residency variant.
- **HIPAA** — `https://hipaa-api.jotform.com` — Gold/Enterprise compliance endpoint.

## OpenAPI
Jotform does not publish a downloadable OpenAPI/Swagger document at a stable public URL as of 2026-05-08; pipeline did not retrieve a spec into `openapi/`. SDKs exist for PHP, Python, Java, Node.js, Ruby, Go, C#, Scala, Android and iOS.
